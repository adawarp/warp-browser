# Warp machine #

### What is this repository for? 
Warp machine is a browser based application to control Warp OS.

![yellowLine](docs/assets/images/yellowLine.png)

### How do I get set up? ###

```sh
$ git clone path
$ cd path
$ npm install
$ npm start
```

![yellowLine](docs/assets/images/yellowLine.png)
